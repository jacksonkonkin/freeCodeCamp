Basic HTML and HTML5

1. Say Hello to HTML Elements

HTML elements are formed with opening and closing tags like <h1> </h1> which in this case is a main heading.

Much like the h1 heading element, there are h2, h3, and all the way to h6 elements to indicate the different levels of

3. Inform with the Paragraph Element

<p> elements are for paragraphs. As a convention, html tags are written in lowercase. Most text blocks are written
inbetween 'p' tags.

4. Fill in the Blank with Placeholder Text

lorem ipsum text is usually used as placeholder text. This tradition has been going on since the 16th century.

5. Writing HTML in plain text

To write HTML in plain text use the & with either a lt or gt to write < and >. Then write whatever between them to
make it look like an HTML element.

6. Uncomment HTML

To leave a comment in HTML, use <!-- to open comment and --> to close the comment. Comments are useful for describing
your code.

Example:

<!-- Comment Goes Here -->

7. Introduction to HTML5 Elements

HTML5 has many descriptive tags including: main, header, footer, nav, video, article, and others. Thses tags give structure
to your HTML and make it easier to read. They also help with Search Engine Optimization (SEO) and accesibility.

8. Add Images to Your Website

Add images with the img element. Point to a specific image's URL with the src attribute. An image must have an alt attribute.
The alt attribute is used for screen readers and improves the accesibility if image fails to load. If image is purely decorative
then an empty alt attribute is best practice.

Example:

<img src="https://www.freecatphotoapp.com/your-image.jpg" alt="This is the description">.

9. Link to External Pages with Anchor Elements

With the use of the anchor element 'a', content outside of the webpage can be linked. 'a' elments need a destination using
the href attribute. They also need anchor text. Another attribute for the anchor elemnts is the target, which specifies if
the link opens up the webpage on a different tab or not. The attribute with _blank opens up the link in a new tab and if no
target is specified, it opens in the same tab.

Example:

<a href="https://www.freecatphotoapp.com" target="_blank">This is a link using the anchor element

This is a link using the anchor element

10. Link to Internal Sections of a Page with Anchor Elements

Anchor elements can also be used to create internal links to jump within a webpage. To accomplish this, assign the href
attribute a value of '#id', id being the element you want to jump to's id attribute. An id attribute uniquely describes
an element.

Example:

<a href="#main-title">Jump to the top of webpage</a>

Jump to the top of webpage

12. Make Dead Links Using the Hash Symbol

You can use the attribute href with a value of '#' to create a dead link. This can be useful when building a website without
knowing where to link to yet. It can also be useful when using JavaScript. When a link is clicked with '#' as the href value
it jumps to the top of the current page the link is clicked, so if there is a target attribute with the value of '_blank' it will
open the same page in a new tab.

The button element with a value of 'submit' in the type attribute will send the data from your form to the URL

You can make form fields required by using the 'required' attribute within the input element.

22. Create a set of Radio Buttons

Use radio buttons when the question being asked can only have one answer out of many. Radio buttons are a type of input.
When using radio buttons it is good practice to wrap the input element inside of a label element so the radio button is
associated with the label.

All related radio buttons should have a name attribute that is the same for each button. This will create a group of
radio buttons. This will ensure that when one radio button is selected, the rest of the radio buttons with the same
name attribute get de-selected. This ensures only one answer may get picked.

Example:

<label>
 <input>
</form>

It is best practice to use the 'for' attribute on the label element with a value that matches the id of input element.
This will allow assistive technologies to create a linked relationship between the label and related input element.

Example:

<label for="indoor">
 <input id="indoor" type="radio" name="indoor-outdoor">Indoor
</label>

23. Create a Set of Checkboxes

If we want people to select more than one answer we give them all the checkboxes they want. Some fun facts about checkboxes

much the same as the radio except square and can click many.

Example:

 <input id="Republican" type="checkbox" name="politics">Republican

24. Use the value attribute with Radio Buttons and Checkboxes

When a form gets submitted, the data is sent to the server and includes entries for the options selected. Inputs
of type radio and checkbox report their values from the value attribute.

For Example:

<label for="indoor">
 <input id="indoor" value="indoor" type="radio" name="indoor-outdoor">Indoor
</label>
<label for="outdoor">
 <input id="outdoor" value="outdoor" type="radio" name="indoor-outdoor">
</label>

Here, you have two radio inputs. When the user submits the form with the indoor option selected, the form data will include
the line: indoor-outdoor=indoor. This is from the name and value attributes of the "indoor" input. If you omit the value attribute, the submitted form data uses the default value, which is on. In this scenario, if the
user clicked the "indoor" option and submitted the form, the resulting form data would be indoor-outdoor=on, which is
not useful. So the value attribute needs to be set to something to identify the option.

25. Check Radio Buttons and Checkboxes by Default

You can set a checkbox or radio button to be checked by default using the checked attribute.
To do this, just add the word checked to the inside of an input element.

For example:

<input type="radio" name="test-name" checked>

The div element, also known as a division element, is a general purpose container for other elements.
The div element is probably the most commonly used HTML element of all.
Just like any other non-self-closing element, you can open a div element with %ltdiv> and close it on another line with
</div>.

27. Declare the Doctype of an HTML Document

At the top of every document, you must specify which version of HTML you are using with the !DOCTYPE element. HTML is
constantly changing and some web browsers may use different versions. The most common version when this was written is HTML5.

Example:

<!DOCTYPE html>

The rest of your code must be written between html tags

Example:

<!DOCTYPE html>
<html>
 Your Website
</html>

28. Define the Head and Body of an HTML Document

To further organize the html, use head and body elements within the html element. Any markup with information about
the page goes in the head and then content goes in the body. There are other elements that can go inside the head including
the link, meta, title, and style elements.

Example:

<!DOCTYPE html>
<html>
 <head>
  <meta/>
 </head>
  <div>
  </div>
 <body>
 </body>
</html>

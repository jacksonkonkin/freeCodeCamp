Basic HTML and HTML5

HTML is a markup language that uses special syntax or notation to describe
the structure of a webpage to the brower. HTML elements usually have opening
and closing tags that surrond and give meaining to content. For example, 
different elements can describe text as a heading, paragraph, or list item.

In this course, I will build a cat photo app to learn some of the most 
common HTML elements - the building blocks of any webpage.
